About Whitelist Rules
Rule Types
  • Domain: Allow emails from specific domains (e.g., senate.gov, ssa.gov)
  • Sender: Allow emails from specific email addresses (e.g., info@trusteddomain.com)
  • Country: Allow emails from specific countries using 2-letter ISO codes (e.g., US, GB, CA)
Pattern Matching
  • Exact: Must match exactly (e.g., "senate.gov" only allows senate.gov)
  • Wildcard: Matches patterns (e.g., "*.gov" allows all .gov domains)
Note: Whitelist rules bypass spam filtering. Global whitelist rules (superadmin only) apply to all client domains.
