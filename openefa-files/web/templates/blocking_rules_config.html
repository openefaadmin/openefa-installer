About Blocking Rules
Rule Types
  • Domain: Block emails from specific domains (e.g., spam.com)
  • Sender: Block emails from specific email addresses (e.g., spammer@example.com)
  • Country: Block emails from specific countries using 2-letter ISO codes (e.g., RU=Russia, CN=China, KP=North Korea, IR=Iran, NG=Nigeria)
Pattern Matching
  • Exact: Must match exactly (e.g., "spam.com" only blocks spam.com)
  • Wildcard: Matches patterns (e.g., "*.cn" blocks all .cn domains)
Note: Blocking rules are processed in order of priority. Be careful when creating wildcard rules as they may block legitimate emails.
