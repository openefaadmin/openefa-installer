About Domain Management
What is a Client Domain?

Client domains are the email domains that this SpaCy system processes. Each domain can have its own blocking rules, whitelists, and security configurations.

Adding Domains
  • Domain: The email domain (e.g., example.com, yourcompany.net)
  • Client Name: A friendly name for the organization
  • Active/Inactive: Control whether the domain is currently being processed
Note: After adding a domain here, you may also need to:
  • Add it to the HOSTED_DOMAINS list in app.py for report filtering
  • Configure Postfix routing rules
  • Set up DNS records (MX, SPF, DKIM, DMARC)
